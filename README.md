# RCxn - Comparative Concepts Analysis

This repository contains analysis of comparative concepts in adjective and adverb constructions.

## 📁 Project Structure

- `analysis/` - Main analysis code and outputs
  - Python scripts for RDF parsing, similarity analysis, and visualization
  - Generated outputs (plots, networks, reports)
  - Data files and documentation

## 🚀 Quick Start

```bash
cd analysis
python run_analysis.py
```

## 📊 Analysis Overview

This project analyzes similarity relationships between comparative concept constructions using:
- RDF data parsing
- Multi-level similarity metrics (slot, construction, CC type levels)
- Network analysis and visualization
- Interactive plots and reports

## 📋 Branches

- `main` - Stable version
- `cc-adjective-adverb-network` - Latest analysis with fixed visualizations

See `analysis/README.md` for detailed documentation.
